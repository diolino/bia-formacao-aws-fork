#################################################################
#### Passo a passo (do clone ao primeiro push)

1) git clone https://github.com/diolino/bia-formacao-aws-fork.git
2) cd nome-do-repositorio

3) Se já não tiver configurado a autenticação
git config --global user.name "Seu Nome"
git config --global user.email "seu-email@exemplo.com"

4) Cria um arquivo para testar
echo "# Meu Projeto" >> teste.txt

5) Verifica o status 
git status

6) Adiciona o novo arquivo ao stage
git add .


7) Faz o commit no repositório logal
git commit -m "Primeiro commit: adiciona estrutura inicial do projeto"

8) Verifica se está conectado ao repostitório remoto
git remote -v

9) Faz o Push
#Se o branch principal for main
git push -u origin main
#
Se for master:
git push -u origin master
